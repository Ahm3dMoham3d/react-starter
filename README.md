# React-Starter

Welcome to the React-Starter project! This template is designed to help you quickly set up a new React application with all the essential dependencies you need for development.

## Features

- Fast development setup with [Vite](https://vitejs.dev/)
- UI components with [Shadcn UI](https://shadcn.dev/)
- State management with [React Redux](https://react-redux.js.org/)
- Routing with [React Router](https://reactrouter.com/)
- Form handling with [React Hook Form](https://react-hook-form.com/)
- Utility-first CSS with [Tailwind CSS](https://tailwindcss.com/)

## Prerequisites

Make sure you have the following installed on your machine:

- [Node.js](https://nodejs.org/) (v14 or later)
- [npm](https://www.npmjs.com/) (comes with Node.js)
